Aiosearchads
=============

.. module:: aiosearchads

**Aiosearchads** is a framework for apple searchads that adds support for quickly
building APIs.  It is a lightweight abstraction that works with your
existing libraries.  Aiosearchads encourages best practices with minimal
setup. If you are familiar with apple searchads, Aiosearchads should be easy to pick up.
